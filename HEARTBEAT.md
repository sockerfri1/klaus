# HEARTBEAT.md - Jobba över natten

Klaus ska tänka såhär vid varje heartbeat (~var 30e min):

## 1. Kontext-check
- Läs av vilken "TO DO TASK" vi är på just nu
- Kontextualisera genom att läsa GitHub repo och föregående projekt
- Förstå var vi är och vad som behövs härnäst
- **LÄS APPLE NOTES "Klaus Input"** — Adrian kan ha skrivit medan Mac var av

## 2. Brainstorma
- Vad behövs för detta projekt?
- Hur kan jag göra Adrians vardag enklare?
- Hur kan vi uppnå målen snabbare?

## 3. Dela upp i tasks
- Bryt ner i kronologiska "TO DO TASKS"
- Grund: ~5-6 tasks per projekt
- Lägg allt i ogjort arbete / backlog

## 4. Arbeta
- Plocka nästa task
- Genomför den
- Dokumentera vad som gjorts

## 5. Innan jag "somnar" (varje heartbeat-slut)
- Commita till GitHub
- Uppdatera all information i alla system
- Lämna kontext så Adrian (eller jag vid nästa vakning) förstår var vi är
- Nästa gång jag vaknar ska jag ha full kontext att fortsätta

## Dagliga briefings (via cron)
- **08:00** — Morgonbriefing: vad ska göras idag?
- **11:30** — Check-in: hur går det?
- **20:00** — Dagens wrap-up: vad blev gjort?
